SaveIT(프로젝트 진행 중)

SaveIT은 React SPA 구조로 개발한 저축 챌린지 기반 자산 관리 웹앱입니다. 자동 저축, 소비 리포트 시각화, 구글 로그인 연동 등 사용자 편의성과 실용성을 중점으로 구현했습니다.

  • TYPE React 기반 싱글 페이지 애플리케이션 (SPA)
  • FEATURE 자동 저축 챌린지, 소비 패턴 시각화, 구글 계정 로그인 기능
  • FEATURE 자동 저축 챌린지 기능, 소비 리포트 시각화, Firebase 인증 연동
  • TOOLS React, Styled-components, GitHub, SourceTree, Figma, Notion, Slack
  • PROCESS 기능별 역할을 분담해 기획·디자인·개발을 진행하고, Context 구조와 백엔드 API 연동을 통해 SPA를 구현
  • ROLE Figma를 활용한 UI 시안 제작, Styled-components 기반 컴포넌트 스타일링, 반응형 Context 구조 설계

MINI

  • TYPE 반응형 웹 (Responsive Web)
  • PAGE 브랜드 메인 페이지 리디자인
  • FEATURE 슬라이드 콘텐츠, 브랜드 이미지 강조 UI
  • TOOLS HTML, CSS, jQuery, Slick Slider, AOS.js
  • PROCESS 브랜드 조사 → 콘텐츠 구조 설계 → 반응형 구현
  • ROLE UI 기획, 디자인 설계, 반응형 마크업 및 퍼블리싱 구현
